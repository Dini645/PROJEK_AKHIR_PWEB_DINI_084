Website ini merupakan proyek web pribadi saya yang menampilkan profil singkat saya sebagai mahasiswa Universitas Ahmad Dahlan, termasuk informasi identitas, hobi, keahlian, serta harapan saya di bidang teknologi. Website ini dibuat menggunakan HTML, CSS, JavaScript, dan PHP, serta dijalankan melalui server lokal XAMPP. Dengan tema pink sebagai warna favorit, proyek ini menjadi media latihan dan representasi diri saya dalam portofolio digital.
